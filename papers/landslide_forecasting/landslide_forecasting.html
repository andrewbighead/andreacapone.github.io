Machine Learning and Explainable Deep Learning for Forecasting Landslide Deformation using EGMS Time Series

Sadegh Madadi, Andrea Capone, Shilpa Srinivasareddy, Diego Di Martire, Aniello Murano

Applied Soft Computing (preprint) • 2026

Download PDF SSRN page

Abstract

Reliable forecasting of slow-moving landslide deformation remains challenging due to non-stationarity, noise, missing observations (gaps), and the cumulative nature of satellite-based displacement products. In this study, the time axis is defined on a fixed 12-day cadence, so the one-step forecast corresponds to a consistent 12-day horizon. Raw cumulative sequences are transformed into robust learning inputs through point-wise normalization using statistics computed exclusively from the training interval, local linear detrending, slope-encoded auxiliary channels, and outlier suppression. A hybrid model combining multi-scale convolutional filters, stacked Bidirectional LSTM layers, and a self-attention block is trained in an incremental next-step formulation, reducing drift accumulation while enabling short-term deformation forecasting. Evaluation on a strictly held-out temporal interval demonstrates stable generalization across heterogeneous kinematic regimes. Gradient-based attribution indicates that the network primarily relies on recent temporal states, yielding physically interpretable relevance patterns. Overall, the proposed framework provides a reproducible baseline for large-scale ground-motion forecasting and supports early evolution tracking of slow-moving landslide processes.
